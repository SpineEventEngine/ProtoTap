public class Paths

    Constants and utility functions for calculating directory and file names used by ProtoTap Gradle and protoc plugins.

    This object may also be useful for users of ProtoTap to obtain its output.

      • outputRoot

         final Path outputRoot(String buildDir)

        Obtains the path to the root directory into which ProtoTap puts all the intercepted files.

        By convention, it is a subdirectory named TARGET_DIR placed under $buildDir/resources/test of a Gradle project.

        Parameters:
        buildDir - The path to the build directory of the project.

      • outputFile

         final String outputFile(String buildDir, String shortFileName)

        Obtains a full path to the file with given shortFileName under the given buildDir of a Gradle project.

        By convention, the file is placed under the TARGET_DIR directory, which is created under $buildDir/resources/test/.

        Parameters:
        buildDir - The path to the build directory of the project.
        shortFileName - The name of the file under the outputRoot directory.
